Lookup Sample - Operators

Demonstration Links

Note that range operators use alphabetic/lexicographic ordering for string fields, so 'a14' < 'a2'

'a1*': account number starts with 'a1' '!a1': account number is NOT 'a1' 'a1?': account number is 'a1' followed by another character
'a1|a6': account number is 'a1' or 'a6' 'fr* && *k': fiscal officer user id starts with 'fr' AND ends with 'k' 'a1..a3': account number is in the range 'a1' to 'a3'
'>a3': account number greater than 'a3' '>=a3': account number is greater than or equal to 'a3' '<a3': account number is less than 'a3'
'<=a3': account number is less than or equal to 'a3' 'NULL': account name is null '!NULL': account name is not null

About KRAD

Kuali Rapid Application Development (KRAD) is a framework that eases the development of enterprise web applications by providing reusable solutions and tooling that enables developers to build in a rapid and agile fashion. KRAD is a complete framework for web developers that provides infrastructure in all the major areas of an application (client, business, and data), and integrates with other modules of the Rice middleware project. In future releases, KNS will be absorbed into and replaced by KRAD.
