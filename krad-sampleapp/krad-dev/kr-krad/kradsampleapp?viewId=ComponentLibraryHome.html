Elements

Basic elements that can be used within groups such as links, actions, and messages.

Controls

Controls are used to input data and include components such as checkboxes, text input, and select controls, and are used with Fields.

Fields

Fields provide labels, instructional text, constraints, and/or binding information for the content they wrap, and are primarily used to display or allow input of information.

Containers

Containers can contain fields and other containers, and provide a layout for the items they contain.

Layout Managers

Layout Managers are used by some containers to control how fields and the containers the layout behave and look on a page.

Widgets

Widgets are provide javascript enhanced component configuration , or expand a component's base features, which allow for a variety of responsive interfaces.

Navigation

Various navigation components are available to navigate between pages and views of a KRAD application.

Collection Features

Collections are able to display their information and manipulate their information in a variety of ways by using various features and options, such as totalling and row details.

Validation

Validation is controlled by constraints and a wide variety of contraints are available to control the input allowed in a Field, and errors are shown immediately.

Client Responsiveness

The client can respond to a variety of actions taken by the user, show content in response to actions, and even perform asynchronous server requests.

General Features

Additional features of KRAD not covered by one of the other categories, but are part of the KRAD feature set.

About KRAD

Kuali Rapid Application Development (KRAD) is a framework that eases the development of enterprise web applications by providing reusable solutions and tooling that enables developers to build in a rapid and agile fashion. KRAD is a complete framework for web developers that provides infrastructure in all the major areas of an application (client, business, and data), and integrates with other modules of the Rice middleware project. In future releases, KNS will be absorbed into and replaced by KRAD.
