Build something cool.

Kuali Rapid Application Development(KRAD) is our UI framework that provides the infrastructure to make rich, robust, and interactive web applications easily with Kuali Rice.

Development made easy

Rapid Application Development

Why "rapid"? Common solutions for technical problems that each application doesn't have to solve every time such as a common look & feel, accessibility, security, and more.

Open Source Frameworks

There's no need or benefit to building everything from the ground up. Like all of Rice we utilize the best open source frameworks where it makes sense. With KRAD we're using Spring, jQuery, Freemarker and (in the near future) JPA.

Easier Development

KRAD creates a common way of developing the user interface and the Rice team maintains the open source libraries and maintenance challenges that often plague homegrown software applications.

Modern Rich Web UIs

The ability to create modern, rich web UIs with improved functionality and efficiency. This is that web 2.0 stuff that everyone talks about and expects a modern application to possess. This very application is built KRAD!

Integration with Rice

Built in integration with all of Rice: workflow, identity management, etc. Greatly enhances the usefulness of KRAD by covering the majority of functionality an institution needs.

Community

A development model that engages and encourages community involvement, contribution, and shared solutions. Check out the links below to get involved!

About KRAD

Kuali Rapid Application Development (KRAD) is a framework that eases the development of enterprise web applications by providing reusable solutions and tooling that enables developers to build in a rapid and agile fashion. KRAD is a complete framework for web developers that provides infrastructure in all the major areas of an application (client, business, and data), and integrates with other modules of the Rice middleware project. In future releases, KNS will be absorbed into and replaced by KRAD.
